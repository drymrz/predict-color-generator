# Tensorflow - Color Generator

## How to setup:

```
git clone https://github.com/drymrz/tensorflow-predict-color-generator.git
cd learn-color-generator
pipenv install
pipenv shell
pip install -r requirements.txt
```

## Requirements:

- matplotlib
- numpy
- pandas
- seaborn
- tensorflow
